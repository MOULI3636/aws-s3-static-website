About This Project

Mouli Banerjee successfully deployed this static website on Amazon Simple Storage Service (S3). The bucket is configured with static website hosting, a public bucket policy allowing GetObject, and uses index.html as the entry point. This demonstrates complete understanding of AWS S3 hosting.

Why S3 Static Hosting?

High Performance

Edge-optimized infrastructure and automatic replication across AZs delivers low latency globally. Content is delivered faster than traditional servers.

Secure by Default

IAM policies, bucket ACLs, and block public access features give fine-grained control. Your data is encrypted and protected.

Cost Efficient

Pay only for what you use! No monthly server fees. For low-traffic websites, it costs pennies per month. 99.99% durability.

CI/CD Ready

Integrate with GitHub Actions, AWS CodePipeline, or CLI for automated deployments. Push code → Auto deploy to S3.
